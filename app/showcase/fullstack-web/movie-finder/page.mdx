Movie Finder is a **React-based web application** that allows users to **search for movies, view full details, favorite them, and organize them into lists**. Powered by the **OMDb API and Firebase**, this app delivers a seamless and interactive movie browsing experience.

## **Why Movie Finder?**

I built this project to **experiment with external APIs and Firebase’s real-time database**. The goal was to create an intuitive platform where users could not only search for movies but also **save and organize their favorite picks** into personalized lists.

## **Features**

- ✅ **User Authentication** – Login/Signup with **Email or Google**.
- 🎥 **Search for Movies** – Fetch real-time movie data using the **OMDb API**.
- ⭐ **Favorite Movies** – Save and manage your favorite films.
- 📜 **View Full Plot Details** – Get detailed descriptions for each movie.
- 📂 **Create, Update, Delete Movie Lists** – Organize your watchlists.
- 🔔 **Toast Notifications** – Instant feedback on user actions.

## **Tech Stack**

**📚 Core Packages**

- **firebase** – Authentication, Firestore for storing and querying data.
- **bootstrap** – Responsive UI styling.
- **react-router-dom** – Client-side routing for seamless navigation.
- **react-toastify** – User-friendly toast notifications.

**🔧 Additional Tools**

- **OMDb API** – Fetching movie data.
- **React Hooks** – Efficient state and effect management.
- **LocalStorage** – Persisting user data across sessions.

## **User Experience & Design**

The app is designed with **Bootstrap styling** to ensure a **clean, responsive interface**, making movie discovery and organization effortless.
