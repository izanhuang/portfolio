Peri is an **online e-commerce web application** designed to provide a **scalable and serverless shopping experience**. Built with **React.js, AWS Lambda, API Gateway, and DynamoDB**, this project showcases the power of **serverless architecture** in handling e-commerce operations.

## **Why Peri?**

I created Peri to **explore serverless computing with AWS** and design a full-fledged online shopping platform with:

- A **product landing page** for showcasing items.
- A **persistent shopping cart** that saves user selections.
- **Secure authentication and authorization** using AWS Cognito.

## **Features**

- 🔐 **Secure Login & Signup** – User authentication via **AWS Cognito**.
- 📦 **Product Listings** – Display items dynamically.
- 🛍️ **Persistent Shopping Cart** – Users can add, remove, and modify cart items.
- ⚡ **Serverless Backend** – Powered by **AWS Lambda & API Gateway**.
- 📊 **NoSQL Database** – Uses **DynamoDB** for scalable storage.

## **Tech Stack**

**📚 Core Technologies**

- **React.js** – Frontend framework for building the UI.
- **AWS Lambda** – Serverless functions to handle business logic.
- **API Gateway** – Secure API endpoints for data transactions.
- **DynamoDB** – NoSQL database for storing product and user data.
- **AWS Cognito** – User authentication and authorization.
- **Amplify** – Hosting and deployment of the frontend.

## **User Experience & Design**

The platform is designed with a **modern and responsive UI**, ensuring a **smooth shopping experience** across all devices.
