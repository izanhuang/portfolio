Peri is an **online e-commerce web application** designed to provide a **scalable and serverless shopping experience**. Built with **React.js, AWS Lambda, API Gateway, and DynamoDB**, this project showcases the power of **serverless architecture** in handling e-commerce operations.

## **Why Peri?**

I created Peri to **explore serverless computing with AWS** and design a full-fledged online shopping platform with:

- A **product landing page** for showcasing items.
- A **persistent shopping cart** that saves user selections.
- **Secure authentication and authorization** using AWS Cognito.
