A **simple yet effective online timer** designed to help users **manage work or study sessions** using the **Pomodoro Technique**. This tool ensures **structured focus periods** followed by well-deserved breaks to improve productivity.

## **🔥 Features**

⏳ **Customizable Session & Break Timers**  
🔄 **Auto-Reset Between Cycles**  
🔊 **Sound Notifications**  
🎯 **Minimal & Focused UI**

## **⚙️ Tech Stack**

- **HTML, CSS, JavaScript** – Core frontend technologies
- **React.js** – Component-based UI
- **useState & useEffect** – Timer state management
