A fully functional **web-based calculator** built using **React.js**. Designed to handle **basic arithmetic operations**, this calculator offers a **clean and intuitive UI** for quick calculations.

## **🔥 Features**

- ➕➖✖️➗ **Perform Basic Arithmetic Operations**
- 🖥️ **Responsive & Interactive Design**
- 🎨 **Smooth User Interface with React.js**

## **⚙️ Tech Stack**

- **React.js** – UI components and state management
- **CSS** – Styled for a modern look
- **useState Hook** – Handling calculator logic
