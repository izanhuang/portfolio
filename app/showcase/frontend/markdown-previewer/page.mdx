A **real-time markdown editor and previewer** for writing and formatting markdown with **instant live preview**. Ideal for developers and writers who use markdown in their workflow.

## **🔥 Features**

📝 **Write & Preview Markdown Instantly**  
🔄 **Auto-Update Between Editor & Previewer**  
🎨 **Styled for Readability & Ease of Use**

## **⚙️ Tech Stack**

- **React.js** – UI and state management
- **Marked.js** – Markdown parsing
- **CSS** – Custom styling for clean display
